## Simplifying assumptions about production behaviour
1. the firm produces only a single good
	1. desiscions blir väldigt komplicerade annars
2. firm redan valt vilken produkt de ska producera
	1. vi måste bara välja hur de ska göra det effektivt
3. for the quantity the firm makes, deras mål är att minimera produktionskostnad
	1. cost minimization necessary to maximize profits är en annan assupmtion
4. använder bara två inputs för göra produkt: capital and labour
	1. capital = maskiner raw materials buildings
	2. labour = human workers
	3. lumpar ihop allt för att simplifya?
5. I short run kan inte välja hur mycket capital man använder, men kan justera mängden labour. Kan i long term justera båda
	1. short run = period vilken då minst en input inte kan ändras
	2. fixed inputs kan inte bli changed i short run
	3. variable inputs kan bli changed i short run
6. more input = more output
	1. liknar more is better
7. production har diminishing marginal returns to labour and capital
	1. är mer produktivt att ha en kombination av de
8. kan köpa så mycket capital or labour inputs it wants at fixed market prices
	1. firms små relativt dessa markander, kan köpa utan att ändra priset
9. Om finns well funcitoning capital market så finns ingen budget constraint
	1. skillnad från consumer choice theory. Här kan man istället låna eller issua stock för investering om den är profitable'

## returns to scale
- the amount of change of output in response to a proportional increase in all inputs, dvs öka allt lika mycket
- *constant returns to scale* om öka labour & capital med en multiple ökar quantity of output med samma multiple
	- ex ![[Pasted image 20251115143354.png]]
- *Increasing returns to scale* om ökar mer än proportionately
- *decreasing returns to scale* om ökar mindre än proportionately
- även om marginal products är alltid fallling kan returns to scale vara constant/increasing, eftersom marginal products refererar till en ökning i bara en input
- grafisk representation med isoquants
	- ![[Pasted image 20251115144021.png]]
- ofta constant returns to scale om exempelvis kan bygga en ny fabrik, kan vara increasning returns to scale om man har mycket fixed costs
- kan vara increasing om man har learning by doing
- decreasing är ovanlligt, kan bara göra exakt samma sak igen och få constant
